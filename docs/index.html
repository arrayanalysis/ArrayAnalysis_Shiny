About ArrayAnalysis

ArrayAnalysis aims to make transcriptomic data analyis accessible for everyone. By providing an easy-to-use interface, ArrayAnalysis allows scientists unfamiliar with high throughput data analysis to analyse and interpret transcriptomic experiments. The analysis of both RNA sequencing and microarray-based transcriptomic data are supported. ArrayAnalysis can be easily installed as a desktop app or as an R package. Follow the installation guide and start using ArrayAnalysis today!

ArrayAnalsis is back!

After 10 year, ArrayAnalysis is finally back! In addition to microarray data, ArrayAnalysis now also supports the analysis of RNA sequencing data. Moreover, new analysis workflows and interactive visualizations support the interpretation of transcriptomic experiments as never before. ArrayAnalysis is run locally, so you don't need to worry about uploading private data on a server.
